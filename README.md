# Apex Dashboard

> Stop switching between notes. One page. Everything you need. Memo your thoughts, crush your todos, track your projects — and make it look incredible doing it.

## Features

### Memo
Capture thoughts instantly with a built-in memo pad. Each memo card has a writable textarea — jot down ideas, meeting notes, or daily reflections without leaving your dashboard.

### Todo
Manage tasks with interactive checklists. Add, reorder, drag-and-drop, and check off tasks. A progress bar shows completion percentage at a glance.

### Projects
Organize your vault documents into project cards. Each card links to related notes, displays a cover image, and supports inline document search to add new files quickly.

### Quick Links
Pin your most-used documents to the sidebar for one-click access. Add, remove, and reorder links freely.

### Banner
A customizable banner with an inspirational quote and optional background image. Double-click to edit.

### Drag & Drop
Drag cards between sections to reorganize your workspace. Drag task items within Todo cards to reorder.

### Custom Sections
Need more than the defaults? Add custom sections to organize your workflow your way.

### Recent Documents
The sidebar shows recently edited files with relative timestamps, so you can jump back into your latest work.

## Settings

- **Dashboard file** — customize the file path for your dashboard data
- **Style** — choose from 7 visual themes
- **Language** — English or Chinese interface
- **Recent documents count** — control how many recent files appear

## Installation

### From Obsidian Community Plugins (coming soon)
1. Open Settings > Community Plugins
2. Search for "Apex Dashboard"
3. Click Install, then Enable

### Manual Installation
1. Download the latest release from [GitHub Releases](https://github.com/pandorareads/apex-dashboard/releases)
2. Extract into your vault's `.obsidian/plugins/apex-dashboard/` folder
3. Open Settings > Community Plugins and enable "Apex Dashboard"

## Usage

1. Open the dashboard via the ribbon icon (home icon) or command palette: `Apex Dashboard: Open dashboard`
2. A `dashboard.md` file is automatically created in your vault root
3. All changes are saved directly to the file — it's your data, in plain text

## Compatibility

- Obsidian v0.15.0+
- Desktop and mobile
- All themes work in both light and dark Obsidian modes
